# Product

CodeDefense helps a developer explain an AI-assisted local repository rather than generate more code. It previews a bounded source snapshot, asks three evidence-backed technical questions, permits at most one follow-up per question, and produces an educational understanding report.

The MVP accepts local folders only. It never executes analyzed source code, uses the user’s existing Codex CLI authentication instead of an OpenAI API key, and keeps repository content out of the runtime except for the explicitly confirmed bounded prompt snapshot.
